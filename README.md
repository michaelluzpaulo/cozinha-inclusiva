## Getting Started

#Local Terminal

```bash
npm i
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

# 1. Extrair schema atual do Supabase

npm run db:pull

# 2. Abrir interface visual para explorar dados

npm run db:studio

# 3. Gerar migrações futuras

npm run db:generate

# 4. Aplicar migrações

npm run db:migrate
